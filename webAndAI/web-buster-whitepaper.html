What is Web Buster!

Web Buster is a Chrome Extension (Manifest V3) that transforms any webpage into an interactive retro gaming experience. The extension overlays classic arcade games on top of any website, using the page's DOM elements as obstacles and collectibles‚Äîcreating a unique fusion of modern browsing and nostalgic arcade entertainment.

Intelligence

AI-Powered Game Selection

Automatic Recommendations

AI analyzes webpage structure to select the best-fitting game based on DOM obstacle count, page type detection, element distribution, and interactive elements count.

Smart Scoring System

Snake: Best for scattered layouts and card-based designs
Shooter: Ideal for articles and text-heavy vertical content
Asteroids: Optimized for open space and image-heavy pages
Pac-Man: Perfect for pages with many obstacles and links

Shows AI confidence level and reasoning for game selection, providing transparency in the recommendation process.

Manual Override

Users can manually select any game from dropdown menu, maintaining full control over their gaming experience.
